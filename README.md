# vibe-coding-30days

30-day coding challenge workspace.

## Goal

- Build one small artifact each day.
- Keep outputs lightweight and practical.
- Write short notes for learning and next actions.

## Structure

- `week1`: Python CLI basics
- `week2`: Data and automation scripts
- `week3`: Bots, dashboards, and app prototypes
- `week4`: Final mini-system + retrospective
- `prompts`: Reusable prompt patterns

## Quick Start

1. Open a day file.
2. Fill in TODO sections.
3. Run and log outcomes.
