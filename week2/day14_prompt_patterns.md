# Day 14 Prompt Patterns

## Pattern: Role + Goal + Constraints

```text
You are a [role].
Goal: [clear outcome]
Constraints:
- [constraint 1]
- [constraint 2]
Output format: [format]
```

## Pattern: Step-by-step Builder

```text
Task: [task]
Steps:
1. [step 1]
2. [step 2]
3. [step 3]
Return only [desired output].
```
