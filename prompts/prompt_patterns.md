# Prompt Patterns

## 1) Problem Solver

```text
Role: Senior [domain] assistant
Goal: Solve [problem]
Inputs: [input data]
Constraints: [rules]
Output: [exact format]
```

```text
역할: 시니어 [분야] 어시스턴트
목표: [문제]를 해결하라
입력: [입력 데이터]
제약 조건: [규칙]
출력: [정확한 형식]
```

## 2) Code Refactor

```text
Refactor this code for readability and maintainability.
Keep behavior unchanged.
Add brief comments only where logic is non-obvious.
Return final code only.
```

```text
이 코드를 가독성과 유지보수성을 높이는 방향으로 리팩터링하라.
동작은 변경하지 말 것.
로직이 명확하지 않은 곳에만 간단한 주석을 추가할 것.
최종 코드만 반환할 것.
```

## 3) Learning Coach

```text
Teach me [topic] at beginner level.
Use one practical example.
Then give 3 short exercises with answers.
```

```text
[주제]를 초보자 수준으로 가르쳐줘.
실용적인 예시 하나를 사용할 것.
그런 다음 답변이 포함된 짧은 연습 문제 3개를 제시할 것.
```
