# Prompt Patterns

## 1) Problem Solver

```text
Role: Senior [domain] assistant
Goal: Solve [problem]
Inputs: [input data]
Constraints: [rules]
Output: [exact format]
```

## 2) Code Refactor

```text
Refactor this code for readability and maintainability.
Keep behavior unchanged.
Add brief comments only where logic is non-obvious.
Return final code only.
```

## 3) Learning Coach

```text
Teach me [topic] at beginner level.
Use one practical example.
Then give 3 short exercises with answers.
```
